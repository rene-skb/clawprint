Questions

Do I need OpenClaw?

Yes. Clawprint uses OpenClaw's cron system, sub-agents, memory tools, and channel integrations. You'll need a running gateway.

How technical do I need to be?

Comfortable with a terminal and editing text files. No coding — everything is markdown and JSON. The guide walks you through each step.

Can I customize everything?

It's all files in your workspace. Change times, swap topics, adjust the assessment, edit crons. You own it completely.

What model do I need?

Works with any model OpenClaw supports. Deep Self and research work best on Opus or Sonnet. Everything else runs on anything.

Is this a subscription?

No. One-time purchase. You get everything plus future updates. Your API costs are your own — Clawprint doesn't add any.
