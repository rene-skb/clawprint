# Clawprint Landing Page Redesign Plan
*Synthesized from: fresh eyes review, 9-persona design panel, Scott's reactions. Mar 4, 2026.*

## Core Problem
The page sells the WHY well but fails on WHAT. People don't know what Clawprint actually is within 10 seconds, it feels like 3-4 products bundled together, and the page never makes the case for why this is better than just writing your own SOUL.md.

## Strategic Decisions

### 1. Focus the product
Strip down to TWO things:
- **Personality assessment** (the core value prop)
- **Memory system configuration** (the infrastructure that makes it stick)

**CUT from the page entirely:**
- Bonus routines section (Daily Signal, Morning Letter, Voice Essays, Research Agent)
- These dilute the message and make it feel like a grab bag

### 2. Address "why not DIY?"
The assessment IS the value prop. Most people can't accurately assess their own Big Five, Enneagram, and Zone of Genius. The structured conversation surfaces things self-reflection misses. The calibration doc translates psychology into agent behavior rules. That translation is the hard part nobody wants to do themselves.

### 3. Price honestly
- Drop the $79 crossed out anchor (feels manipulative)
- $29 flat. Simple. Honest.
- No fake urgency

## Page Structure (revised)

### Nav
- Add simple anchor links: "How it works" | "What you get" | "FAQ"
- Scrolls to sections on click

### Hero (KEEP as is, add one clarifying line)
- "Your agent works for you. Imagine if it *knew* you."
- Sub: "How you think. What you're chasing. When to push and when to listen. One conversation, and it does."
- **ADD a one-liner:** "Clawprint is a personality assessment and starter kit for OpenClaw agents." Small, below the sub, before the CTA. Says what it is without killing the vibe.
- **ADD:** "Requires OpenClaw" as a small tag near the CTA so it's visible early.
- Price: just "$29" clean, no crossed-out anchor
- "Set up in 25 minutes" instead of "Set up in an evening"

### Flow Section (MOVE UP, this is the strongest part)
Assessment → Profile → SOUL.md → Agent adapts
- Cut step 4 (Daily Signal example is generic and embarrassing)
- Replace with something that shows the agent actually behaving differently based on calibration
- **Chat bubbles:** Fix the triangle artifacts, match border radius properly, clean up jank
- **Chat content:** Keep the new aspirational version (flow/passion question, not conflict)

### Before/After (KEEP but evaluate)
- "Same agent. Different relationship."
- This and "Here's how it works" might be redundant. Consider merging or cutting one.
- If we keep both, before/after stays above to sell the why, flow section shows the how.

### What You Get (simplified)
- A full psychological profile across validated frameworks
- An agent calibrated to how you actually work
- An upgraded memory system that never forgets
- An agent who's smarter every day, learning from every mistake

### Built on Real Psychology (KEEP, minor edits)
- Fix "Throughline" — people don't know this word. Say "core direction" or "Zone of Genius" instead
- "supported by recent research" (already updated)
- Don't overclaim

### What's Inside (KEEP, simplify)
- Keep the 3 groups but tighter
- Cut Research Agent card
- Memory system cards stay (this is core)

### Set Up Steps (revise timing)
- "Set up in 25 minutes" not "an evening"
- 4 steps stays but rewrite step 4: not "wake up to a different agent" (implies overnight). Something like "Start talking. It's already different."

### FAQ (KEEP, good)
- Accordion pattern is solid
- Make sure OpenClaw requirement is clear here AND near the top

### Bottom CTA (KEEP, simplify)
- "Make it yours." 
- $29, clean
- No crossed-out price

## Interactive Ideas (future, not this pass)
- **Bret Victor inspired:** Interactive profile bars where you drag sliders and the SOUL.md calibration text updates live below. "Move the Agreeableness slider and watch how your agent's feedback style changes." This would be a killer demo.
- **Try 3 questions:** Embedded mini-assessment on the page. Answer 3 questions, get a tiny profile preview. Hook that makes people want the full thing.
- These are V2 features, not blocking launch.

## Design Fixes
- Chat bubble triangles: clean up, match border radius, no jank
- Bonus grid: remove entirely (cutting bonus section)
- Profile bars: slightly thicker (8-10px)
- Overall: the design is strong, just needs the content to match

## Copy Fixes
- Kill "Throughline" on the page (replace with plain language)
- "A memory that never forgets" — memories CAN forget, be honest. "A memory system that holds everything" or similar.
- Don't overclaim what the assessment delivers
- Make the case for assessment > DIY somewhere on the page

3. Cut bonus routines section entirely
4. Fix pricing ($29, no anchor)
5. Fix "set up in 25 minutes"
6. Move flow section higher
